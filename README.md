# evaluacion-usuarios-api
# Evaluación API Usuarios con XMLHttpRequest

## Objetivo de la aplicación

Esta aplicación tiene como objetivo consumir datos desde el endpoint:

https://jsonplaceholder.typicode.com/users

La aplicación utiliza JavaScript, programación orientada a objetos y XMLHttpRequest para obtener, almacenar y administrar la información de usuarios entregada por el web service.

A través de botones HTML, permite ejecutar distintos métodos para consultar y mostrar información específica de los usuarios.

## Funcionalidades

La aplicación permite:

- Obtener datos desde un web service usando XMLHttpRequest.
- Almacenar la información dentro de una clase para su posterior uso.
- Listar los nombres de todos los usuarios.
- Buscar un usuario por nombre y mostrar su información básica:
  - Username  
  - Correo electrónico  
- Buscar un usuario por nombre y mostrar su dirección completa:
  - Calle  
  - Suite  
  - Ciudad  
  - Código postal  
  - Coordenadas geográficas  
- Buscar un usuario por nombre y mostrar su información avanzada:
  - Teléfono  
  - Sitio web  
  - Compañía  
  - Frase clave  
  - BS  
- Listar todas las compañías junto a su frase clave.
- Listar los nombres de los usuarios ordenados alfabéticamente.

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- XMLHttpRequest  
- API REST JSONPlaceholder
